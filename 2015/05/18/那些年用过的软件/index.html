那些年用过的软件

发表于 2015-05-18   |   分类于 效率   |  

绪论

时间就是生命，效率就是金钱。生活如此美好，绝不能浪费世间在无聊的任何事情上，高效的工具是值得拥有的。

那些年这些年，自己用过n多软件，好多都被大浪淘沙，死在前任和后来者之手，但是，不管如何，适合自己需要的才是最好的——而这句话的前提是自己要亲自试过比较过。

正文一：必装软件

Listary

这绝对是老少咸宜，居家旅行必备神器。它的作用就是快递定位文件。
安装后，设置一个快捷件“win+Q”，使用时候，在任何情况下（注意是任何情况下）按这个快捷件，出来一个弹话框，输入文件的名称，名称可以是中文/英文，还可以是中文/英文首字母，比如输入“yd”
！[搜索有道画面]（）
至于其它英文搜索就更不在话下。
通常查找到，然后就可以直接打开了，配合其它软件用的很爽。比如我用的
Directory Opus这个资源管理器，看图的faston等等。
这个软件还有个很爽的地方就是，当你用浏览器保持或者上传文件的时候，一般是直接一步一步的找到位置的，而它可以省去寻找过程，直接搜索出直达位置，省去n多功夫。
这是中国人做的软件，虽然是要收费的，免费也照样能用，偶尔出来的个对话框督促缴费而已。
说真的，这个软件真的值得付费。

Everything

这个软件也是搜索用的，特点是快，快到极致了，反正没有见谁超过它的速度，比起listary还要快好多。
缺点也很多，不支持中文拼音首字母，比如上面的“yd”出来的就不是有道了。
一般用这个来综合整理文件用，比如将分散在各个磁盘的电影统一重新规划放置。平时主要用listary多一点。
同样是免费。

Directory Opus

这是个来自澳洲的windows平台资源管理器，比微软自身的强了x倍。类似的软件有很多，但是这个和Total Commander可以算是两种方向的极端代表。TC适合键盘狂，基本不用鼠标的人，跟windows的资源管理器完全不一样的。而opus就比较适合鼠标一族，绝对要平民得多，可以理解为windows的资源管理器的加强版。opus有很强大的定制，也可以默认，反正我使用这个软件之后，节省了n多时间，推荐每个人都应该用的，尤其是经常跟文档文件打交道的人。这软件比较烦人的就是反破解达到了丧心病狂的地步。
配合listary用起来那叫个爽。行云流水啊。

替补软件比较推荐XYplorer。

Ditto

这是个复制管理软件。windows下每次copy都会替换掉上一次copy的内容，无法重复利用之前的copy。用了这个就可以很随意的查找到甚至几天前的copy了。大大的节省了我的时间。开源免费软件，绝对推荐。

以上四个软件我是必装的，每一个都是用过n多同类软件后筛选出来，省时省力。

正文二：优选考虑

FastStone

这是个看图软件，它不是最快的，也不是最强大的，但是作为一个平时主要看图偶然整理以下图片的软件来说，它是最合适的，看图的体验非常好，而且免费。备选xinview。

ShareX

这是个截图神器，强到爆炸，它最牛逼的地方就是在你截完图后会自动帮你完成后面的步骤，后面的步骤可选非常非常丰富，比如，写博客时候，我要插入某些图片，切换到相应页面，按“ctrl+print screen”，然后回到博客文档按“ctrl+ V”粘贴网址，OK！这过程，它完成了哪些任务呢，它把图传到图床（如photobuck），然后将相应的网址复制到粘贴板中，我就直接可以使用了!!!这是最简单的，这期间，它还可以加水印，保存在本地电脑，备份到dropbox，分享出去等等一系列动作，节省了x倍时间啊！！！
这是免费的啊！

Opera

浏览器真他妈的多，就是没有1个好用的。opera算是综合各种因素，自己用得最多的了。现在还有个Vivaldi，opera的创始人新开炉灶的，可惜还是不怎么好用，不过值得期待。如此怀念当初的opera！

说说插件吧：

1.poket：稍后阅读的代表，心血来潮的时候用一下，其实真的蛮好的。

2.鼠标手势：还是很喜欢的，配合快捷键，行云流水啊。

3.印象笔记·剪藏：用来收藏某篇文章或者某段文字

4.同步书签：永远也不怕丢辛苦收藏的网址了。

5.send to kindle

6.IFFF：这是个牛逼到爆炸的idea，有待完善，值得关注。

7.SwitchyOmega: 代理软件管理者

8.暴力猴 :脚本管理者

Photoshop

简称PS，这软件大概是所有软件中教材最多的吧，强大到没边。完全没对手，就是不知道为啥没人告它垄断呢？！

ABBYY FineReader

ORC光学电脑自动识别图文软件，很好用，时不时的要拍些照片，然后用这个将照片的文字转换成为可以搜索的文字档案，如pdf档。图片清晰度越高，转换的准确率越高，基本95%以上，当然电脑也就越慢了，人越省事。省了n倍加x倍的功夫，文字越多越体现效率。节省出来的时间当然是用来享受生活的啦。

虹吸墨 / notepad++

两个必选一个用来代替windows自带的记事本。
虹吸墨特别适合不会写代码的人，作者还做了好多视频教材摆放到网上，功能强大，绿色还免费。
最喜欢的一个功能就是：将一堆文件/文件夹拖放到虹吸墨那里，瞬间得到所有这些文件的名称。碉堡啦！

DisplayFusion

这是多屏幕时候用的，需要两个或者以上屏幕时候特别好，工作中用excell，省了多少功夫啊！！

正文三：比较个人的

foobar2000 之 shutter版本

最爱的音乐播放器，没有之一。

CorelDRAW

习惯这个了，后来又懒得去学AI，其实AI配合PS使用挺好的。没办法了，反正coreldraw也挺好，就这吧。

Painted

数字绘画专用软件，配合wacome真是好的很，里面的仿真笔刷牛逼的不得了，比ps的笔刷好了n倍。

keyshot

简单利落的3D渲染软件，效果也很强，专业为工业设计制造。

正文四：一些其它的

hexo

目前大概是各种的静态博客生成器中的使用中最省心的了，当然安装配置什么的对非码农来说还是有难度，我折腾了好久，折腾过程中，感触最深的就是，基本没有教材是写给非码农看的，你们这些个码农啊，能不能不要这么以己度人啊，唉。。。。不过搞定安装之后就好办了，写文章也简单。在这过程中学到不少东西，真是不容易啊。

Markdownpad

简单易用的markdown写作软件，用来写个blog还是不错的。表格真烂。。。

Cmder

有了这个，终于不用windows自己的cmd了，连个复制粘贴都没有的cmd赶紧死去！

utorrent

自从BitComet堕落后一直用这个，不过现在也基本不用啦，我不bt很多年了。怀念一下吧。

eDonkey/Xtreme

eDonkey是所有电骡的总称，Xtreme是其中非常出名版本，也是我一直用的版本，虽然他们自身都死了，却活在其了其它地方。 没有经历过bt和电骡疯狂的时代的人，很难理解这种p2p软件的伟大，各种版本的电骡们疯狂的撕裂了传统娱乐大亨的城墙护盾，颠覆了许多与之相关的行业，基本上所有唱片公司都面临不同程度的倒闭危机，迫使所有的人为之倾倒，揭开了一个全新的数字音频时代。实在是有太多值得回忆和思考的了，有机会真是应该写篇长文祭奠一二。

CCleaner

清除垃圾专用

iRead

这是电脑用的阅读器，仿真度满高，视觉效果还行，主要是没有找到更好的，笔记太烂，就没有一个阅读软件的笔记好用的，无语。

PotPlayer & KMPlayer

混着用来看电影，主要是没有找到更好的。

网易云音乐

用来听音乐，能找到些不错的歌单。目前没有没有更好的。

福昕阅读器

pdf专用，比较少看pdf档，所以只好放这里了，其实还是很好用的。

Evernote

说真的，没觉得这个有多好多牛逼，只是习惯了而已，关键其它同类的也不怎么样。只好摆这了。话说微软onenote的手写笔记功能配合绘图板爽得很，以后准备改用这个了。dropbox倒是很牛逼，可惜被和谐，话说这个和谐真是无语，百度360之流的网盘n多不和谐内容还不是好好的。大概是因为这些企业没啥底线，十分听话，随便出卖客户隐私，而dropbox这种不听话的企业只有被墙的份了。

小狼毫

干净利索的拼音输入法。偶尔也微软拼音，话说微软拼音有时候打句子真爽，就是翻页选择很不习惯。

猎豹安全浏览器

专门用来刷火车票用，平时不用。

结论之中场休息

以上都是些我用过又值得推荐和记录的，而有一些我有用但真不值得记录，比如qq，迅雷，office，淘宝旺旺等等，不是说他们不好，而是因为
上面的软件基本是围绕效率而展开的，省时省力省心才是王道——所以呢有不少在用但不应该上来这里的。

这些软件肯定不是唯一的选着，因为还有n多没有用过的软件，而且时代在进步，软件在迭代，所以呢，这不是冠盖定论，而是中场休息，更多更好的美好世界在等着我们去开发，哪天有更爽的，就要搞个续集了。正所谓：

欲知后事如何，期待下回分解！
