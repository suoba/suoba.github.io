腰痛根源

发表于 2018-05-09   |   分类于 中医   |  

前言

余初读黄师之腰痛根源，百思不得其解，缘何其篇不说跌打损伤致使腰痛？须知这是常见之痛也，网查，非余一人不悟也。唯有搁置跳过。

两月后，忽有亲友腰痛不能蹲，坐，痛甚时，起床翻身皆不能也。中/西医皆尝试过，西医谓之腰间盘突出/坐骨神经痛，中医谓之肾虚。初有好转，后加重，而后换医，复如故。兜兜转转竟已半年！何其悲哀也，真是书到用时方恨少啊，余自问读书不下万本，然又有何用？！一怒之下，遍查古今医书之腰痛论，得益现代互联网，能快速遍查诸子百家之医书，方知论腰痛者数不胜数，却皆是知其然不知其所以然也，补肾者论更是十居其七，罕有明鉴者，唯顾金寿《重订灵兰要览》之腰痛论尚可一读也，但也仅限于此。

若论知其根源者，当属医圣张仲景，可惜后人不晓其深意，直至黄元御，诉说之于《四圣心源》腰痛根源篇也。余多次翻阅，方晓其真意。顿悟之下，初买药2盒劝说其服之,7天内药尽,竟然好至七八成，翻身起床蹲坐弯腰行走皆可已，尚不能久坐及负重弯腰。千回百转竟又回到起点，真是百感交集，是以写下此篇以记之。

正文

腰痛根源唯一（本）

常人木生于水，水暖木荣，生发而不郁塞，所以不痛。

腰痛病因有五（标）
1. 湿气盛 : 所以土湿，木郁，水泛，下寒也。

2. 肾虚 : 常人当上虚下实，肾在下，虚则水泛滥，土湿而木郁，是以病作。

3. 淤血 : 若有淤血或干血在水木之间，气血凝滞，管路堵塞，流水不畅，则水不能生木，是以痛作也。

4. 跌打损伤 : 若其人有伤在腰间，常常有淤血在其中，是以痛作也。

5. 肝木之为病 : 若其干枯，不得生发，水积而泛滥，是以病作也。

病源惟一，病因有五。

五种病因，传承转变，皆会导致同一种根源，腰痛根源。知其根，晓其由，断无不治之理也。

问诊之时，当细问其病之起因及其人房事，劳作，居所，湿寒等工作生活环境和习惯，无非外因或内因也。

所谓病因有五，非五中选一，可能有二三不止，或有并发，或有诱发，或有转变，用药期间当间隔复查诸病症，随症下药，用药不能随意“从一而终”。无论其转变如何，谨记其病源惟一，病因有五。追根溯源，总不离其中，下方用药，当治其病因而保其根源，八九不离十也。真传一句话，假经十万卷！

后记

这篇文章没有具体的方药：

一是，如果你看得懂上面的腰痛根源，不用我说你也知道怎么下方用药了；如果你没有看懂，说明你不够资格来治疗腰痛病，若有方，必定会“误导”你以为方固定，病稍有不同，你就不会用了，很容易导致你乱用药而不自知，害人伤己，有违我的本意。

其二，就算我出方用药，也不过在伤寒论中选出一二方罢了，并无出彩之处，顺便一提，我用来治疗亲人的药方就是出自伤寒论，而且这个方在伤寒论中原文并未提及治疗腰痛，但是我就用了，也治好腰痛了，说到这里估计明晓此篇文章的人已经知道我用啥伤寒论方了。

其三，万一有土豪愿意找我问诊，我好出个狠价钱啊，否则真对不起这半年的辛苦查阅，就2000好了（什么？你嫌贵啊，身为穷鬼就 应该像我这样好好读书，把服务留给土豪享受），不包药！还不能包好，因为有的 人是生活工作习惯引起的腰痛，病好后不改变习惯也没用，照样复发！！你要是日操夜干的，神仙也救不了你的腰肾啊！
